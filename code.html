JavaScript Code Description

  1. answerExtract() Function ( My Own Code ) : I used it in Quiz Page to Calculate The Score of Student/Your Answers.
  2. wlmSite() Function ( My Own Code ) : to Print The Score of Student Answers and his name and his ID and say Welcome .
  3. setStyle() Function ( My Own Code ) : to Change The Color of Student Answer and set it Red if it's Wrong ,and Green if it's Correct.
  4. disable() Function ( My Own Code ) : To disable the selection feature in the input tags and select tags with option tags when click on an any input or select any option.
  5. enable() Function ( My Own Code ) : To reactivate (enable) the selection feature in the input tags and select tags with option tags when click on Clear Button.
  6. Sticky Header Function ( Free Code | Embedded in All Pages) : To make the Header Sticky when you Scroll in The y-axis.
  7. ListTable() Function ( My Own Code | Embedded in CV-FORM Page) : To Display Personal Information in Tabular Format.
  8. showCode() Function ( My Own Code ) : To Print Out The Code Page Content.
